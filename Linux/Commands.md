# Basic Linux Commands

- Create Directories

```mkdir <dir name>```

- Create Files

```touch <file name>```

- Copying File

```cp <source> <destination>```

- Copy Directory

```cp -r <Source> <Destination>```

- Move File/Directory

```mv <Source> <Destination>```

**Note:** This command can be used to rename as well

- Remove file

```rm <File Name>```

- Remove directory

```rm -r <Directory Name>```
